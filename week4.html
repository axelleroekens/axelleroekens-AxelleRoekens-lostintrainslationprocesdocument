Eerste keer HTML

In deze les hebben we kennisgemaakt met HTML en CSS. Tegen volgende week moeten we met ChatGPT een code laten schrijven in HTML en CSS, zodat we ons ontwerp kunnen zien en de eerste stappen in Visual Studio Code kunnen zetten.

Ik heb met een screenshot en de prompt die we kregen de code gemaakt en in Visual Studio Code gezet, zodat ik kon zien wat er van gemaakt is.

Voor figma heb ik mijn schermen aangepast en het contrast tussen de verschillende delen aangeduid met een hiërarchie.Het witte en blauwe contrasteert beter dan het zwarte want dat is heel donker. Ook heb ik bij de ene pijltjes toegevoegd voor welke richting het perron is van het moment dat je dat scherm ziet want soms is het moeilijk om te weten naar waar je moet als je het station niet kent. Achteraf bleek dat wat chaotisch te zijn en werkte het niet zo goed kreeg ik als feedback dus heb ik dat later in de app weg gelaten.
